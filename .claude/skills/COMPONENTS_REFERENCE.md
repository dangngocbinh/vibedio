## ðŸ“ 0. TIKTOK CAPTION COMPONENT

**File**: `src/components/captions/TikTokCaption.tsx`

### Use Cases
- Word-by-word highlighted captions (TikTok/Reels style)
- Auto-generated from voice timestamps
- 8 pre-designed themes for different content types
- Optimized for readability and engagement

### Quick Start
In `script.json`:
```json
{
  "subtitle": {
    "theme": "clean-minimal",
    "position": "bottom"
  }
}
```

### Available Themes
1. **clean-minimal** (default) - White pill background, professional
2. **gold-bold** - Bold yellow text, attention-grabbing
3. **drop-green** - Green highlight, dynamic animation
4. **red-box** - Red background box, emphasis
5. **impact-yellow** - Extra large, maximum impact
6. **neon-glow** - Glowing neon, tech/futuristic
7. **story-elegant** - Serif font, emotional/storytelling
8. **minimal-dark** - Simple dark text, aesthetic

### Props Reference

```typescript

  words: CaptionWord[];       // âœ… REQUIRED - Auto from voice.json timestamps

  // THEME (choose one approach)
  theme?: string;             // Theme name (e.g., 'clean-minimal')

  // POSITION
  position?: 'top' | 'center' | 'bottom' | 'bottom-high';
  // Default: 'bottom'

  // OPTIONAL OVERRIDES
  font?: string;              // Override theme font
  highlightColor?: string;    // Override theme highlight color
  fontSize?: number;          // Override theme font size
  letterSpacing?: string | number;  // Override letter spacing

  // ADVANCED (usually not needed)
  startOffset?: number;       // Start time offset in seconds
  frame?: number;            // Manual frame control

```

### OTIO Integration

**IMPORTANT:** Captions are auto-generated by `subtitle_generator.py`. You don't need to manually create caption clips.

If you need to create manually:
```python
clip = otio.schema.Clip(
    name="Caption",
    metadata={
        "remotion_component": "TikTokCaption",
        "props": {
            "words": [  # From voice.json timestamps
                {"word": "Hello", "start": 0.0, "end": 0.5},
                {"word": "world", "start": 0.5, "end": 1.0}
            ],
            "theme": "clean-minimal",
            "position": "bottom"
        }
    }
)
```

### Best Practices

âœ… **DO:**
- Use `clean-minimal` as default (works for 80% of content)
- Keep position at `bottom` for TikTok/Reels standard
- Match theme to content mood
- Let automatic generation handle it

âŒ **DON'T:**
- Don't use center position by default
- Don't mix multiple themes in one video
- Don't override theme styling randomly
- Don't manually create caption clips (use auto-generation)

**Educational Video:**

**Viral Facts:**

